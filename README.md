## Требования

- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/) (обычно входит в состав Docker Desktop)

## Быстрый запуск

1. **Скачайте архив** с репозитория или клонируйте его:

   ```bash
   git clone https://github.com/Dsquar-gys/VerstaTest.git
   cd VerstaTest
  
2. Запустите compose файл

    Обратите внимание, что в целях простоты запуска .env уже находится в репозитории

    ```bash
    docker compose up

3. Откройте в браузере

    Фронтенд: http://localhost:3000
   
    Бэкенд API: http://localhost:8080/scalar
